============
Installation
============


Stable release
--------------

To install Flood Area Segmentation, run this command in your terminal:

.. code-block:: console

    $ pip install floodsegment

This is the preferred method to install Flood Area Segmentation, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Flood Area Segmentation can be downloaded from the `Gitlab repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://gitlab.com/draconus21/floodsegment

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://gitlab.com/draconus21/floodsegment/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Gitlab repo: https://gitlab.com/draconus21/floodsegment
.. _tarball: https://gitlab.com/draconus21/floodsegment/tarball/master
